## Features

* Converts all types of design nodes into clean Unity UGUI components
* Generates fully responsive layouts for desktop and mobile
* Automatically organizes assets into folders (Fonts, Images)
* Supports gradients, solid colors, and image fills
* Auto-creates Canvas based on design frame, with option for custom target Canvas
* Adds automatic scrolling for designs exceeding screen size
* Converts all text elements to TextMeshPro
* Optimized download of complex shadows, blends, and effects as images
* Adds generated pages directly into the current Unity Scene
* Supports custom font assignment via settings panel
* Extremely fast re-generation without re-downloading assets
* Imports only necessary assets—no unused images or styles
* Works entirely inside Unity with no external dependencies

## Project Setup

### Step 1: Open Converter Settings

* Navigate to **Tools → FigmaDesign To UI Converter** in the Unity Editor to open the settings panel.

### Step 2: Configure Advanced Settings

* Set the **Output Directory** where all generated files will be saved
* Adjust optional checkboxes based on your workflow:

  * ✅ Refetch FigmaDesign — re-download even if previously fetched
  * ✅ Replace Images — overwrite existing exported images
  * ✅ Use Default Fonts — disable to apply custom fonts
  * ✅ Target Canvas — specify which canvas to place generated UI
  * ✅ Generate One Canvas Per Page — for multi-page designs
  * ✅ Download Text as Image — export text layers as images if needed
* Select your preferred **Image Format** and **Image Scale**
* Click **Convert FigmaDesign** to generate your Unity UI instantly

### Step 3: Optimize Game View

* Adjust the aspect ratio in **Game View** to match your original design for best responsiveness
